🔧 PROBLEMAS RESOLVIDOS (DETALHES)
────────────────────────────────────────────────────────────────────────────

┌─ Problema 1: TypeError em Models
│  Erro: descriptor '__getitem__' requires 'typing.Union'
│  Causa: SQLAlchemy 2.0.38 incompatível com Python 3.14 type hints
│  Solução: Remover Mapped[Optional[...]] de 45+ campos
│  Status: ✅ RESOLVIDO
│  Arquivo: TECH_REFERENCE_SQLALCHEMY.md

├─ Problema 2: Alembic usando SQLite
│  Erro: sqlite3.OperationalError: near "EXTENSION": syntax error
│  Causa: alembic/env.py caindo em fallback SQLite
│  Solução: Reconfigurar para settings.DATABASE_URL direto
│  Status: ✅ RESOLVIDO
│  Arquivo: backend/alembic/env.py

├─ Problema 3: PowerShell Parse Errors
│  Erro: TerminatorExpectedAtEndOfString
│  Causa: Emojis em scripts PowerShell (encoding UTF-16LE)
│  Solução: Substituir 🌐🔗⏹️ por [REDE][ENDPOINTS][STOP]
│  Status: ✅ RESOLVIDO
│  Arquivos: scripts/run-dev-server.ps1, run-frontend-dev.ps1

├─ Problema 4: PostgreSQL Service Offline
│  Erro: asyncpg.ConnectionDoesNotExistError
│  Causa: Serviço PostgreSQL não iniciado
│  Solução: Criar Iniciar_PostgreSQL.bat com automação
│  Status: ✅ RESOLVIDO
│  Arquivo: Iniciar_PostgreSQL.bat

└─ Problema 5: Banco Não Inicializado
   Erro: Migrations não aplicadas, tabelas inexistentes
   Causa: Setup incompleto
   Solução: Menu automático em Iniciar_PostgreSQL.bat
   Status: ✅ RESOLVIDO
   Arquivo: TROUBLESHOOTING.md

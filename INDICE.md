# Indice do FROTAS

## Operacao

| Tarefa | Como acessar |
|---|---|
| Central Operacional | `FROTA_Iniciar.bat` |
| Diagnostico | `powershell -File .\Diagnostico.ps1` |
| Backup automatico | `Configurar_Backup_Automatico.bat` |

## Opcoes principais da Central

| Opcao | Uso |
|---|---|
| Iniciar stack dev | Backend + frontend |
| Preparar PostgreSQL local | Banco, migrations e seed |
| Status | Portas, PIDs e logs |
| Parar ambiente | Encerrar processos |
| Backup manual | Gerar backup local |
| Publicar porta 80 | Publicacao local |
